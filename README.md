\# Robot Framework Automation Project



\## Overview

This project contains automated test scripts built using Robot Framework and Selenium.



\## Features

\- Login Automation

\- Product Selection Automation

\- UI Validation



\## Tech Stack

\- Robot Framework

\- SeleniumLibrary

\- Python



\## Project Structure

/tests

&#x20;   login\_test.robot

&#x20;   product\_test.robot



/resources

&#x20;   Resource.robot



\## How to Run

1\. Install dependencies:

&#x20;  pip install robotframework

&#x20;  pip install robotframework-seleniumlibrary



2\. Run test:

&#x20;  robot tests/



\## Reports

After execution, reports are generated in HTML format.
